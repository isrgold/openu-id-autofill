

# 🚀 תוסף כרום להזנת מספר זהות אוטומטית

**Open University ID Autofill Chrome Extension**

תוסף כרום זה נועד לפשט ולזרז את תהליך ההתחברות לאתר **האוניברסיטה הפתוחה** 🎓 על ידי שמירה מקומית ואוטומטית של מספר הזהות שלך והזנתו בעמוד ההתחברות.

---

## 🌟 תכונות עיקריות

* 🔒 **שמירה מקומית ומאובטחת** – מספר הזהות נשמר רק בדפדפן שלך ולא נשלח לשום שרת.
* ⚡ **הזנה אוטומטית בעמוד ההתחברות** – בכל כניסה לאתר האוניברסיטה הפתוחה, התוסף ימלא עבורך את שדה מספר הזהות.
* 🖥️ **חוויית שימוש פשוטה** – התקנה חד־פעמית, ללא צורך בהקלדה מחדש.
* 🧩 **קל להתקנה ולניהול** – תוסף קל, מהיר ומבוסס קוד פתוח.

---

## 📋 דרישות

* דפדפן **Google Chrome** (גרסה עדכנית מומלצת)
* גישה לדף ההתחברות של האוניברסיטה הפתוחה

---

## 🔧 התקנה

### 1. הורדת קבצי התוסף

* לחץ על כפתור **Code** בעמוד הפרויקט ובחר **Download ZIP**
* חלץ את הקבצים לתיקייה במחשב שלך

### 2. הפעלת מצב מפתחים ב-Chrome

* פתח את Chrome והיכנס לכתובת:

  ```
  chrome://extensions/
  ```
* הפעל את **Developer mode** בפינה הימנית העליונה

### 3. טעינת התוסף

* לחץ על **Load unpacked**
* בחר את התיקייה שבה נמצאים קבצי התוסף

### 4. זהו! 🎉

התוסף יופיע ברשימת ההרחבות שלך ויתחיל לפעול אוטומטית.
